# CMCA Session Player — Releases

Session zip bundles and APK releases for the CMCA Session Player Android app.

## Downloads

Go to [Releases](../../releases) to download:
- **manifest.json** — session catalog (the app fetches this automatically)
- **Session zips** — individual session bundles (downloaded on-demand by the app)
- **APK** — Android app installer (coming soon)
